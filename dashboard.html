// Tournaments

//tournament-create

Create a new tournament. Sets the game, format (Swiss, Single/Double Elim, Round Robin), max teams, ruleset, and results channel.

name — Tournament name
game — OW2, Marvel Rivals, or Destiny 2
format — Swiss / Single Elim / Double Elim / Round Robin
max_teams — (optional) Max number of teams
ruleset — (optional) Rules text
results_channel — (optional) Channel ID for results
● Pro — Admin

//tournament-end

End an active tournament and post final standings to the results channel.

tournament_name — Name of the tournament to end
● Pro — Admin

//tournament-cancel

Cancel an active tournament. Optionally provide a reason that gets posted to the channel.

tournament_name — Name of the tournament
reason — (optional) Reason for cancellation
● Pro — Admin

//register

Register yourself for an active tournament. Requires your in-game tag and the game you're registering for.

tournament_name — Tournament to register for
game_tag — Your BattleTag, Bungie ID, or username
game — Game you're registering for
● Anyone

//team-create

Create a team for a tournament. You become the team captain. Other players can be added with /roster-add.

team_name — Your team name
tournament_name — Tournament to create the team for
● Anyone

//roster-add

Add a player to your team roster. Must be a registered player. Only the team captain can add players.

player — @mention the player to add
tournament_name — Tournament name
● Team Captain

//report-score

Report your team's Swiss match result. Both captains must report. If scores match, the result is submitted for admin confirmation. If they conflict, an admin is notified to resolve the dispute.

tournament_name — Tournament name
your_score — Your team's score (maps won)
opponent_score — Opponent's score (maps won)
● Team Captain

//ask-td

Ask the AI Tournament Director a question about rules, disputes, standings, or anything tournament-related. AuraAI will respond based on the tournament's ruleset.

question — Your question
tournament_name — Tournament name
● Anyone

//scrim-schedule

Schedule a scrim or find opponents to practice against. AuraAI will post your availability and help coordinate matchups.

game — Game for the scrim
availability — When you're available
notes — (optional) Additional info
● Pro

// FACEIT Match System

//match-start

Kicks off a FACEIT-style match. Creates a private thread for both teams, posts the full ruleset, and immediately begins the Map 1 hero ban phase. The team that sets up the lobby bans second.

tournament — Tournament name
team1 — First team name (lobby captain's team by default)
team2 — Second team name
lobby_team — (optional) Which team is setting up the lobby
mode_rotation — (optional) Comma-separated mode order e.g. Control,Flashpoint,Hybrid,Escort,Push
● Tournament Admin

//ban-hero

Submit your hero ban for the current map. Rules enforced automatically: you cannot ban the same role twice in a map, and a hero can only be banned once per match. Use inside your match thread.

hero — Hero name to ban (e.g. Mercy, Genji, Reinhardt)
● Team Captain

//match-code

Share the custom game match code with both teams inside the match thread. The code is posted cleanly so all players can join.

code — The custom game match code
● Team Captain

//map-result

Report which team won the current map. The bot updates the series score, determines the next mode in the rotation, and prompts the losing team to pick the next map.

winner — Name of the team that won the map
● Either Captain

//map-pick

Losing team picks the next map from the current mode's map pool. After picking, the ban phase begins — the team that picked bans second.

map — Map name to play next (must be in the current mode pool)
● Losing Team Captain
